## 📖 What it does
- **Search by username** – fetch all public repos + open issues count.  
- **Search by topic** – one‑click buttons for JavaScript, HTML, CSS – get featured repos.  
- **Dive into issues** – click any repo to see all open issues with direct GitHub links.  

## ✨ Features
- 🔥 Clean, responsive UI (works on mobile/desktop)  
- 🐙 Real‑time data from GitHub API  
- ⚡ No backend – pure HTML/CSS/JS
